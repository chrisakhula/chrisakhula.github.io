# Cleophas Ouma Digital Hub

Personal portfolio site for Cleophas Ouma, focused on ERP systems, ICT operations, reporting workflows, and technology-led business improvement.

## GitHub Pages

To publish this site at `https://chrisakhula.github.io`, the repository on GitHub must be named `chrisakhula.github.io`.

## Run locally

1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`
3. Build for production with `npm run build`

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
